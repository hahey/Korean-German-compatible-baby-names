# Korean-German compatible baby names - 한국어와 독일어로 통용되는 아기이름

## Introduction

### Why? 왜 한독통용 아기이름을 찾았나요?

As a Korean living in Germany, I thought that it would be nicer if my baby could have a name that
both works in Korean and in German than having two given names, one in Korean and another in German,
especially people tend to choose only one name and use that one, and I can already foresee the
future that the Korean one is likely to be forgotten.

재독 한국인으로서 제 아기에게 한국어로도 독일어로도 통하는 이름을 짓는게 한국이름 하나, 독일이름
하나 따로따로 짓는 것 보다 낫다는 생각이 들었어요. 이름이 두개라면 결국 하나만 부를텐데 독일에
살다보면 아무래도 한국이름은 잊혀질 것 같다는 느낌이 들어서요.

### How? 어떻게 리스트를 만들었나요?

Although there are certain standards or common rules how to find Korean given names, Korean given
names are more flexible and can vary more than given names in the most western countries, also
likely due to the fact that Korean family names are more similar to each other than family names in the
most western countries. So, I went through a bit more than 5,000 german baby names in a few
different lists on internet and checked if their pronunciations could be also regarded as Korean baby
names. Mostly I follow the German version of pronunciations if the English or other versions of
pronunciations do not match but I also made quite some exceptions for convenience. It's not close to an exclusive
list but probably still the longest list that you can find on internet in 2025. Please feel free to
send a pull request or send me an email if you want to add more names in the list.

독일 아기이름에 비해 한국 아기이름이 좀 더 다양하기 때문에 한국어 아기이름 리스트 대신 독일어
아기이름 리스트를 쭉 훑어가며 한국어 아기이름과 비슷하게 발음될 수 있는지를 확인해서 리스트를
구성하였습니다. 이 리스트를 위해서 약 5,000개가 넘은 독일 아기 이름을 여러개의 사이트를 참고해서 쭉
확인했어요. 독일어 발음이 다른 나라 (예를 들어 영어식 발음)과 다른 경우 대부분 독일어 발음을
기준으로 하였는데 편의에 따라 꽤 많은 예외도 두고 있습니다. 이 리스트에 없는 이름 중에서도 한독- 또는
한?(다른나라)-통용 되는 이름을 아시면 편하게 Pull Request나 이메일을 주시면 리스트에 추가하도록 하겠습니다.
아직 부족한 점이 많은 리스트이지만 제가 인터넷에서 본 리스트 중에서는 현재 2025년 기준으로 가장 긴
리스트이기도 합니다.

### Gender? 성별

I happenend to take quite a traditional approach for listing baby names based on only traditional two genders
and unfortunately there are only a few gender neutral names in the list. I respect all the other gender concepts
and also wish to add many more neutral names and to restructure the list not based on two genders
anymore in the future.

어쩌다 보니 남자 아기 이름과 여자 아기 이름으로 분류를 하게 되었는데 추후에 좀 더 중성적인 아기
이름을 추가하고 리스트를 성별과 관계없이 구성할 수 있도록 바꾸려고 노력중입니다.
